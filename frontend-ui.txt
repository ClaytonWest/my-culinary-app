Culinary AI UI Revamp - Implementation Plan                                               
                                                                                           
 Overview                                                                                  
                                                                                           
 Redesign the frontend to a "Minimal Rail" Gemini-style layout with Soft Sage color        
 palette, ambient video header, and modernized chat components.

 - Recipe Book prominent in sidebar (frequent visit)

 - Mobile suggestions as stacked pills (Gemini-style)
